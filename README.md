# VueStore

## 点击右上角克隆或下载下载完成之后，在vue-demo文件目录打开命令窗口输入命令： npm install （需提前装好node.js环境，如果没有请自行百度）待npm安装完成之后输入： npm run dev 即可在浏览器看到窗口界面

## 技术栈

+ webpack
+ vue-router
+ vuex


## 功能

+ 新闻资讯
  - 新闻资讯列表
  - 新闻资讯列表详情
  - 新闻评论功能
+ 图片分享
  - 图片列表
  - 图片缩略图功能
+ 商品购买
  - 经典商品列表布局
  - 商品详情页面
  - 添加购物车
+ 购物车功能
